Detailed Description

SNixieTempoNote defines note as pair of 2 numbers:

  1. note frequency
  2. note duration in parts of the second.

Tempo '2' means 1/2 sec, tempo '8' means 1/8 sec. This way of definition needs only 3 bytes per note and is easy for translating Notes to digital representation.
