Creates object representing display consisting of 1 - 6 Nixie Tubes

Parameters
[in]driver- K155ID1 driver to control all Nixie Tubes (yes, it is the only one - cost reduction)
[in]tubePins- array, containing pin numbers corresponding to controlled tube's Anods
[in]mapTable- default pin mapping for all NixieTubes (refer to NixieTube class)
[in]maxTubes- 1 to 6
