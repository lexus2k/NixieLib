§ render()

void NixieDisplay::render ( )

This method must be called in loop function as often as it's possible.

Warning
If you miss call to this function for a time much more than CHANGE_INTERVAL, defined in nixie_tube.h, this may cause neon gas to flash inside nixie tube (this is bad). Thus, if you need to do some time consuming actions, use powerOff() before and powerOn() after actions.
